## Props

| Prop        | Type                | Default    | Description                                        |
|-------------|---------------------|------------|----------------------------------------------------|
| `direction` | `"row" \| "column"` | `"column"` | Dirección de los elementos de la pila.             |
| `reverse`   | `boolean`           | `false`    | Invierte la dirección de los elementos de la pila. |
