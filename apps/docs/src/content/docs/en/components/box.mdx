Box is the most basic layout primitive. It renders a plain element with the classes and attributes you pass to it, making it useful as a small wrapper when you need structure without built-in styling.

## Basic usage

By default, `Box` renders as a `div`. Use utility classes to control spacing, sizing, color, and layout directly from the component.

<Tabs>
  <TabItem label="Preview">
    <CodePreview>
      <Box class="p-4 w-full bg-(--palette-solid) text-(--palette-contrast)">
        This is the Box
      </Box>
    </CodePreview>
  </TabItem>
  <TabItem label="Code">
    ```astro
    ---
    import {Box} from "@liko-ui/astro";
    ---

    <Box class="p-4 w-full bg-(--palette-solid) text-(--palette-contrast)">
      This is the Box
    </Box>
    ```
  </TabItem>
</Tabs>

## Examples

### Composition

Use `Box` to compose larger UI blocks from smaller layout pieces. It can wrap media, text, and nested containers without adding behavior of its own.

<Tabs>
  <TabItem label="Preview">
    <CodePreview>
      <Box class="max-w-md border border-gray-700 rounded-xs overflow-hidden">
        <Image
          class="w-full h-50 object-cover object-center"
          src={compositionData.imageUrl}
          alt={compositionData.imageAlt}
          width="100%"
          height="200px"
        />
        <Box class="p-4 space-y-2">
          <Box class="flex items-center gap-2">
            <Box class="flex justify-between items-center gap-2">
              <p>{compositionData.rating} ({compositionData.reviewCount})</p>
            </Box>
          </Box>
          <p className="font-medium text-white">
            {compositionData.title}
          </p>
          <Box class="flex items-center gap-2 text-gray-400">
            {compositionData.formattedPrice} • {compositionData.beds} beds
          </Box>
        </Box>
      </Box>
    </CodePreview>
  </TabItem>
  <TabItem label="Code">
    ```astro
    ---
    import {Box} from '@kori/ui';
    ---

    <Box class="max-w-md border border-gray-700 rounded-xs overflow-hidden">
      <Image
        class="w-full h-50 object-cover object-center"
        src={compositionData.imageUrl}
        alt={compositionData.imageAlt}
        width="100%"
        height="200px"
      />
      <Box class="p-4 space-y-2">
        <Box class="flex items-center gap-2">
          <Box class="flex justify-between items-center gap-2">
            <p>{compositionData.rating} ({compositionData.reviewCount})</p>
          </Box>
        </Box>
        <p className="font-medium text-white">
          {compositionData.title}
        </p>
        <Box class="flex items-center gap-2 text-gray-400">
          {compositionData.formattedPrice} • {compositionData.beds} beds
        </Box>
      </Box>
    </Box>
    ```
  </TabItem>
</Tabs>

## Polymorphic

Use the `as` prop to change the rendered HTML element while keeping the same `Box` styling and attributes.

<Tabs>
  <TabItem label="Preview">
    <CodePreview>
      <Box as="section">
        This is a Box rendered as a section
      </Box>
    </CodePreview>
  </TabItem>
  <TabItem label="Code">
    ```astro ins=/as="section"/
    ---
    import {Box} from '@kori/ui';
    ---

    <Box as="section">
      This is a Box rendered as a section
    </Box>
    ```
  </TabItem>
</Tabs>
