Stack provides a layout primitive for arranging related elements in a single direction. Use it when children should share the same flow and spacing without manually managing flex direction classes on each layout.

## Basic usage

By default, `Stack` arranges its children vertically. Apply sizing and spacing classes directly to the stack and its children to control the final layout.

## Props

Use these props to change the stack direction or reverse the visual order of its children.

| Prop         | Type      | Default | Description                                |
|------------- |-----------|---------|--------------------------------------------|
| `horizontal` | `boolean` | `false` | Makes the stack items horizontal.          |
| `reverse`    | `boolean` | `false` | Reverses the direction of the stack items. |
