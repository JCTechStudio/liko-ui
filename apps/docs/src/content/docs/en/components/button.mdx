Button provides an interactive control for actions, submissions, and navigation-like triggers. It includes visual variants, sizing, loading states, and polymorphic rendering for cases where a button needs to use another HTML element.

## Basic usage

By default, `Button` renders a native `button` element with the medium size, round shape, and solid fill.

<Tabs>
  <TabItem label="Preview">
    <CodePreview>
      <Button>Button</Button>
    </CodePreview>
  </TabItem>
  <TabItem label="Code">
    ```astro
    ---
    import {Button} from '@liko-ui/astro';
    ---

    <Button>
      Button
    </Button>
    ```
  </TabItem>
</Tabs>

## Variants

Use variants to adjust the visual treatment of a button while keeping the same interaction behavior.

### Shape

Use `shape` to control the button radius, from sharp corners to fully circular edges.

<Tabs>
  <TabItem label="Preview">
    <CodePreview>
      <Button shape="sharp">Button</Button>
      <Button>Button</Button>
      <Button shape="circle">Button</Button>
    </CodePreview>
  </TabItem>
  <TabItem label="Code">
    ```astro ins=/shape="(?:sharp|circle)"/
    ---
    import {Button} from '@liko-ui/astro';
    ---

    <Button shape="sharp">
      Button
    </Button>

    <Button>
      Button
    </Button>

    <Button shape="circle">
      Button
    </Button>
    ```
  </TabItem>
</Tabs>

### Fill

Use `fill` with a `palette` to choose how strongly the button color appears across solid, subtle, outlined, and text-like treatments.

## States

Button states communicate availability and progress while preserving consistent sizing and layout.

### Disabled

Use `disabled` to prevent interaction when an action is unavailable.

<Tabs>
  <TabItem label="Preview">
    <CodePreview>
      <Button disabled>Button</Button>
    </CodePreview>
  </TabItem>
  <TabItem label="Code">
    ```astro ins="disabled"
    ---

    ---

    <Button disabled>
      Button
    </Button>
    ```
  </TabItem>

### Loading

Use `loading` to disable the button and show progress while an action is running. Add `loadingText` or change `spinnerPlacement` when the loading label should differ from the default content.

<Tabs>
  <TabItem label="Preview">
    <CodePreview>
      <HStack class="gap-4">
        <Button loading></Button>
        <Button loading>
          Loading button with hidden long text
        </Button>
        <Button loading loadingText="Loading...">
          Loading button with hidden long text
        </Button>
        <Button loading loadingText="Loading..." spinnerPlacement="end">
          Loading button with hidden long text
        </Button>
      </HStack>
    </CodePreview>
  </TabItem>
  <TabItem label="Code">
    ```astro ins=/loading(?:Text="Loading...")?|spinnerPlacement="end"/
    ---
    import {Button, HStack} from '@liko-ui/astro';
    ---

    <HStack class="gap-4">
      <Button loading></Button>

      <Button loading>
        Loading button with hidden long text
      </Button>

      <Button loading loadingText="Loading...">
        Loading button with hidden long text
      </Button>

      <Button loading loadingText="Loading..." spinnerPlacement="end">
        Loading button with hidden long text
      </Button>
    </HStack>
    ```
  </TabItem>
</Tabs>

## Polymorphic

Use the `as` prop when the button should render as another element, such as an anchor for link-style navigation.

## Props

Use these props to control the rendered element, visual variants, layout behavior, and loading state.

| Prop               | Type                                                                | Default     | Description                                                  |
|--------------------|---------------------------------------------------------------------|-------------|--------------------------------------------------------------|
| `as`               | `astroHTML.JSX.Element`                                             | `"button"`  | Polymorphic tag used to render the component.                |
| `shape`            | `"round" \| "sharp" \| "circle"`                                    | `"round"`   | Border radius style.                                         |
| `fill`             | `"solid" \| "tonal" \| "surface" \| "outline" \| "clear" \| "text"` | `"solid"`   | Visual fill style.                                           |
| `size`             | `"xs" \| "sm" \| "md" \| "lg" \| "xl"`                              | `"md"`      | Button size scale.                                           |
| `block`            | `boolean`                                                           | `false`     | Makes the button take full available width.                  |
| `palette`          | `Palette` (all tailwind colors)                                     | `undefined` | Palette color key (for example: `blue`, `emerald`, `rose`).  |
| `disabled`         | `boolean`                                                           | `false`     | Disables the button.                                         |
| `loading`          | `boolean`                                                           | `false`     | Enables loading state and disables interaction while active. |
| `loadingText`      | `astroHTML.JSX.Element`                                             | `undefined` | Custom content shown while loading.                          |
| `loadingSpinner`   | `astroHTML.JSX.Element`                                             | `undefined` | Custom spinner node used during loading state.               |
| `spinnerPlacement` | `"start" \| "end"`                                                  | `"start"`   | Spinner position when `loadingText` is present.              |
