Stack proporciona un primitivo de layout para organizar elementos relacionados en una sola dirección. Úsalo cuando los hijos deban compartir el mismo flujo y espaciado sin gestionar manualmente las clases de dirección flex en cada layout.

## Uso básico

Por defecto, `Stack` organiza sus hijos en vertical. Aplica clases de tamaño y espaciado directamente al stack y a sus hijos para controlar el layout final.

## Props

Usa estas props para cambiar la dirección del stack o invertir el orden visual de sus hijos.

| Prop         | Type      | Default | Description                                        |
|--------------|-----------|---------|----------------------------------------------------|
| `horizontal` | `boolean` | `false` | Coloca los elementos de la pila en horizontal.     |
| `reverse`    | `boolean` | `false` | Invierte la dirección de los elementos de la pila. |
