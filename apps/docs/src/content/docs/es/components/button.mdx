Button proporciona un control interactivo para acciones, envíos y disparadores similares a la navegación. Incluye variantes visuales, tamaños, estados de carga y renderizado polimórfico para los casos en los que un botón necesita usar otro elemento HTML.

## Uso básico

Por defecto, `Button` renderiza un elemento `button` nativo con tamaño mediano, forma redondeada y relleno sólido.

<Tabs>
  <TabItem label="Preview">
    <CodePreview>
      <Button>Button</Button>
    </CodePreview>
  </TabItem>
  <TabItem label="Code">
    ```astro
    ---
    import {Button} from '@liko-ui/astro';
    ---

    <Button>
      Button
    </Button>
    ```
  </TabItem>
</Tabs>

## Variantes

Usa las variantes para ajustar el tratamiento visual de un botón manteniendo el mismo comportamiento de interacción.

### Shape

Usa `shape` para controlar el radio del botón, desde esquinas rectas hasta bordes completamente circulares.

<Tabs>
  <TabItem label="Preview">
    <CodePreview>
      <Button shape="sharp">Button</Button>
      <Button>Button</Button>
      <Button shape="circle">Button</Button>
    </CodePreview>
  </TabItem>
  <TabItem label="Code">
    ```astro ins=/shape="(?:sharp|circle)"/
    ---
    import {Button} from '@liko-ui/astro';
    ---

    <Button shape="sharp">
      Button
    </Button>

    <Button>
      Button
    </Button>

    <Button shape="circle">
      Button
    </Button>
    ```
  </TabItem>
</Tabs>

### Fill

Usa `fill` con una `palette` para elegir qué tan presente aparece el color del botón entre tratamientos sólidos, sutiles, con borde y de texto.

## Estados

Los estados del botón comunican disponibilidad y progreso mientras mantienen un tamaño y layout consistentes.

### Disabled

Usa `disabled` para impedir la interacción cuando una acción no esté disponible.

<Tabs>
  <TabItem label="Preview">
    <CodePreview>
      <Button disabled>Button</Button>
    </CodePreview>
  </TabItem>
  <TabItem label="Code">
    ```astro ins="disabled"
    ---
    import {Button} from '@liko-ui/astro';
    ---

    <Button disabled>
      Button
    </Button>
    ```
  </TabItem>
</Tabs>

### Loading

Usa `loading` para desactivar el botón y mostrar progreso mientras una acción está en curso. Añade `loadingText` o cambia `spinnerPlacement` cuando la etiqueta de carga deba diferir del contenido por defecto.

<Tabs>
  <TabItem label="Preview">
    <CodePreview>
      <HStack class="gap-4">
        <Button loading></Button>
        <Button loading>
          Loading button with hidden long text
        </Button>
        <Button loading loadingText="Loading...">
          Loading button with hidden long text
        </Button>
        <Button loading loadingText="Loading..." spinnerPlacement="end">
          Loading button with hidden long text
        </Button>
      </HStack>
    </CodePreview>
  </TabItem>
  <TabItem label="Code">
    ```astro ins=/loading(?:Text="Loading...")?|spinnerPlacement="end"/
    ---
    import {Button, HStack} from '@liko-ui/astro';
    ---

    <HStack class="gap-4">
      <Button loading></Button>

      <Button loading>
        Loading button with hidden long text
      </Button>

      <Button loading loadingText="Loading...">
        Loading button with hidden long text
      </Button>

      <Button loading loadingText="Loading..." spinnerPlacement="end">
        Loading button with hidden long text
      </Button>
    </HStack>
    ```
  </TabItem>
</Tabs>

Usa la prop `as` cuando el botón deba renderizarse como otro elemento, como un enlace para navegación.

      Button
    </Button>

## Props

Usa estas props para controlar el elemento renderizado, las variantes visuales, el comportamiento de layout y el estado de carga.

| Prop               | Type                                                                | Default     | Description                                                                  |
|--------------------|---------------------------------------------------------------------|-------------|------------------------------------------------------------------------------|
| `as`               | `astroHTML.JSX.Element`                                             | `"button"`  | Etiqueta polimórfica usada para renderizar el componente.                    |
| `shape`            | `"round" \| "sharp" \| "circle"`                                    | `"round"`   | Estilo del radio del borde.                                                  |
| `fill`             | `"solid" \| "tonal" \| "surface" \| "outline" \| "clear" \| "text"` | `"solid"`   | Estilo visual de relleno.                                                    |
| `size`             | `"xs" \| "sm" \| "md" \| "lg" \| "xl"`                              | `"md"`      | Escala de tamaño del botón.                                                  |
| `block`            | `boolean`                                                           | `false`     | Hace que el botón ocupe todo el ancho disponible.                            |
| `palette`          | `Palette` (all tailwind colors)                                     | `undefined` | Clave de color de la paleta (por ejemplo: `blue`, `emerald`, `rose`).        |
| `disabled`         | `boolean`                                                           | `false`     | Desactiva el botón.                                                          |
| `loading`          | `boolean`                                                           | `false`     | Activa el estado de carga y deshabilita la interacción mientras está activo. |
| `loadingText`      | `astroHTML.JSX.Element`                                             | `undefined` | Contenido personalizado que se muestra durante la carga.                     |
| `loadingSpinner`   | `astroHTML.JSX.Element`                                             | `undefined` | Nodo de spinner personalizado usado durante el estado de carga.              |
| `spinnerPlacement` | `"start" \| "end"`                                                  | `"start"`   | Posición del spinner cuando `loadingText` está presente.                     |
